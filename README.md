# Vicsek Model

Minimal local implementation of the two-dimensional Vicsek model with:

- periodic boundary conditions
- orientation-colored particles
- velocity arrows
- animated visualization with Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python minimal_vicsek.py
```

Edit the parameters at the bottom of `minimal_vicsek.py` to explore different regimes.
